Proposta
Plataforma de descoberta e inventário automatizado de dados pessoais para apoio à conformidade com a LGPD em pequenas e médias empresas.

Escopo MVP
Scanner para arquivos estruturados locais.
Upload de CSV/XLSX.
Identificação automática de dados pessoais (CPF, nome, email, telefone, endereço).
Classificação de risco.
Dashboard simples.
Geração de relatório de adequação a LGPD em PDF.

Fluxo do usuário
Usuário faz upload de arquivo: CSV, XLSX
